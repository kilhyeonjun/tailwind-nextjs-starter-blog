# 항해99 플러스 백엔드 코스 6기 참여

2024년 8월 9일, 나는 항해99 플러스 백엔드 코스 6기에 합류하게 되었다. 이번 코스에서 나는 Java를 주 언어로 선택했으며, 그 이유는 다음과 같다:

1. Java는 한국의 백엔드 개발 분야에서 널리 사용되는 언어다.
2. 많은 기업들이 자사의 기술 스택에 맞는 개발자를 선호하기 때문에, Java 실력을 갖추면 개발자로서의 경쟁력을 높일 수 있다.
3. 대학 시절과 첫 직장에서의 Java 경험을 바탕으로, 더 심도 있는 학습을 통해 복잡한 비즈니스 로직과 대규모 트래픽을 다루는 능력을 키우고 싶었다.

> [!NOTE]
> 이번 코스에서는 Spring Boot를 활용해 백엔드 개발의 기초부터 심화까지 학습할 예정입니다.

## 사전 스터디 계획

항해99 플러스 백엔드 코스 6기의 본 과정은 9월 21일 토요일부터 시작된다. 그 전까지 총 3개의 사전 스터디가 준비되어 있다:

1. 언어 스터디
2. 프레임워크 스터디
3. 방법론 스터디

나는 이 세 가지 스터디 모두에 참여하기로 결정했으며, 특히 프레임워크 스터디에서는 조장 역할을 맡게 되었다.

## 프레임워크 사전 스터디

8월 13일, 우리 조는 항해 게더타운에서 모여 프레임워크 사전 스터디의 방향을 정했다. 이 스터디는 항해에서 제공하는 가이드를 기반으로 진행될 예정이며, 주요 내용은 다음과 같다:

- 사용 프레임워크: Spring Boot
- 주요 과제: 게시판 만들기

> [!TIP]
> 프레임워크 사전 스터디의 목표:
>
> 1. Spring Boot를 이용한 CRUD 기능의 REST API 구축
> 2. JWT를 활용한 인증 및 인가 구현
> 3. JPA 연관관계를 이용한 회원과 게시글 간의 관계 구현

이번 사전 스터디를 통해 Spring Boot를 활용한 백엔드 개발의 기초를 다지고, 본 과정에서 좋은 성과를 낼 수 있도록 최선을 다할 것이다.

이번 항해99 플러스 백엔드 코스를 통해 Java와 Spring Boot를 깊이 있게 학습하고, 실제 산업에서 요구되는 백엔드 개발 능력을 갖추어 나의 개발자 경력을 한 단계 발전시키고자 한다.
이는 단순히 기술 습득을 넘어, 복잡한 비즈니스 로직을 다루고 대규모 서비스를 설계할 수 있는 능력을 키우는 과정이 될 것이다.
